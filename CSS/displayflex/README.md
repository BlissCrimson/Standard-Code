# Display Flex

## Grundlagen:

Um mit Display Flex nuzen zu k√∂nnen, muss man dem jeweiligem Element den Enstprachendem Style geben.

## Code

### Display Flex vergeben

```css
.d-flex {
  display: flex;
}
```

### Element Vertical Zentrieren

```css
.d-flex-v-c {
  display: flex;
  justify-content: center;
}
```

### ELement Horizontal Zentrieren

```css
.d-flex-h-c {
  display: flex;
  align-items: center;
}
```

### Element Mittig Zentrieren

```css
.d-flex-c-c {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

### Elemente Untereinander Anordnen

```css
.d-flex-column {
  display: flex;
  flex-direction: column;
}
```
